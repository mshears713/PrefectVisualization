# PrefectVisualization Demo - Static Website

This folder contains static HTML visualizations of Python execution pipelines captured using runtime decorators.

## Contents

- **index.html** — Landing page with links to all visualizations
- **module_graph.html** — High-level pipeline visualization showing modules and their relationships
- **task_graph_*.html** — Detailed task-level visualizations for each module

## How to Use Locally

1. Open `index.html` in any web browser
2. Click links to navigate between module and task graph visualizations
3. Interact with graphs to explore execution flow, timing, and data dependencies

```bash
# On Windows
start index.html

# On macOS
open index.html

# On Linux
xdg-open index.html
```

## Deploying to Render

This folder is configured to be deployed as a **static site** on Render.

## HTML Files

The visualization HTML files are generated by the Python application in the `output/` folder. To deploy new visualizations:

1. **Generate visualizations** using the Python application (these are written to `output/` by default)
2. **Copy desired HTML files** into the `viz-demo` folder
3. **Update links** in `index.html` if adding new visualizations
4. **Commit and push** to deploy

Example:
```bash
# Copy latest visualizations
Copy-Item output/module_graph.html viz-demo/ -Force
Copy-Item output/task_graph_*.html viz-demo/ -Force

# Commit and push
git add viz-demo/
git commit -m "Update visualizations"
git push
```

## Troubleshooting

**Graphs not loading?**
- Ensure vis.js CDN is accessible
- Check browser console (F12) for errors
- Verify HTML files are in the `viz-demo` folder

**Links broken?**
- Verify HTML file names match link references in `index.html`
- Check file paths are relative (no leading `/`)

**Mobile view looks wrong?**
- Browser may be caching old CSS; do a hard refresh (Ctrl+Shift+R)
