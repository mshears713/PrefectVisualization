🔍 Visual Python Execution Graph Demo

Interactive Runtime Visualization

This demo showcases runtime execution pipelines captured using Python decorators. Each visualization displays how functions execute, the flow of data between modules, and execution timing.

Two scenarios:

  • ✓ Success Scenario — All modules execute successfully
  • ⚠️ Errors & Warnings Scenario — Shows warnings, failures, and cascading error propagation

💡 How it works: When the Python pipeline executes, the @task and @module decorators capture execution metadata. This data is then visualized as interactive network graphs showing dependencies, timing, and execution flow.

All visualizations are static HTML files powered by vis.js

✓ Works offline • ✓ Mobile-friendly • ✓ No backend required
